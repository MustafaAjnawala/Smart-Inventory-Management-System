### ğŸ› ï¸ **Getting Started**

**Prerequisites:**

*   Node.js
*   MongoDB

**Installation:**

1.  **Backend:**
    ```bash
    cd server
    npm install
    ```

2.  **Web Frontend:**
    ```bash
    cd one-smart-inc-frontend
    npm install
    ```

3.  **Electron Frontend:**
    ```bash
    cd one-smart-inc-frontend-electron
    npm install
    ```

**Running the Application:**

1.  **Start the backend server:**
    ```bash
    cd server
    npm run dev
    ```

2.  **Start the Web application:**
    ```bash
    cd one-smart-inc-frontend
    npm run dev
    ```

3.  **Start the Electron application:**
    ```bash
    cd one-smart-inc-frontend-electron
    npm run dev
    ```
